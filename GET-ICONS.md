# Генерація іконок для PWA

## Онлайн інструменти:

1. **Favicon Generator** - https://realfavicongenerator.net/
   - Завантажте assets/icon.svg
   - Налаштуйте розміри
   - Завантажте готовий пакет

2. **Maskable Icon Editor** - https://maskable.app/editor
   - Для адаптивних іконок Android

3. **Simple Icons** - https://simpleicons.org/
   - Для додаткових іконок

## Мінімальний набір:
- icon-192.png
- icon-512.png  
- favicon.png
